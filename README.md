# MOVE

MOVE is een web-app ontworpen voor momenten waarop je vastzit in een mentaal patroon — overthinking, checken, controleren, oplossen, vermijden.

De app helpt je snel uit je hoofd en terug in contact met het heden. Niet door te analyseren. Door te voelen en te bewegen.

## Kernprincipe

> Als de interactie niet leidt tot actie, heeft het gefaald.

MOVE is geen plek om te blijven. Het is een tool om te bewegen.

## Hoe het werkt

1. Gebruiker opent de app op het moment dat een patroon actief is
2. Typt wat er speelt
3. Krijgt één gerichte interventie
4. Verlaat de app en voert de interventie uit
5. Geeft achteraf feedback (ja/nee)
6. Systeem leert wat werkt

## Wat het niet is

- Geen chatbot
- Geen therapieplatform
- Geen analyse-tool
- Geen plek voor lange gesprekken
